## Quick Start

### Prerequisites

- Docker (version 20.10+)
- Docker Compose (version 2.0+)
- LLM API keys from PPPL (Claude, ChatGPT, Gemini, Perplexity)
- OIDC credentials from IT (for Entra ID authentication)

### Step 1: Configure Environment

```bash
cp .env.template .env
```

Edit `.env` and add your values:

```bash
# REQUIRED: Security keys (auto-generated by setup.sh)
APP_SECRET=your-random-secret-key
ENCRYPTION_KEY=your-random-encryption-key

# REQUIRED: PPPL LLM API Keys
GEMINI_API_KEY=your-gemini-key
OPENAI_API_KEY=your-openai-key
ANTHROPIC_API_KEY=your-anthropic-key
PERPLEXITY_API_KEY=your-perplexity-key

# REQUIRED: OIDC/Entra ID (from IT)
OIDC_CLIENT_ID=pppl-client-id
OIDC_CLIENT_SECRET=pppl-client-secret
```

### Step 2: Start Tales

```bash
docker compose up -d
```

Wait for containers to start (about 30 seconds).

### Step 3: Create Admin Account

```bash
docker compose exec app python scripts/admin/setup_initial_admin.py
```

Follow the prompts. **Save the generated password!**

### Step 4: Import PPPL Brand Data

```bash
./import_data.sh
```

This imports the Princeton Plasma Physics Laboratory brand with:
- 22 monitoring queries
- 17 competitors
- 352 historical responses
- 8 analysis reports

### Step 5: Log In

Open your browser to: `http://localhost:8080`

Log in with your admin credentials from Step 3.

## Updating Brand Data

The included brand data is from February 2026. To get the latest data from production:

1. Connect to the production database (get URL from Railway dashboard):
   ```bash
   export DATABASE_URL="postgresql://..."
   ```

2. Export fresh data:
   ```bash
   python scripts/admin/export_brand_data.py \
     --brand "Princeton Plasma Physics Laboratory" \
     --output data/pppl_brand.json
   ```

3. Re-run the import:
   ```bash
   ./import_data.sh
   ```

## Troubleshooting

### Container won't start

```bash
docker compose logs app
```

### Import fails

1. Verify containers are running: `docker compose ps`
2. Check data files exist: `ls data/`
3. Run import with verbose output:
   ```bash
   docker compose exec app python scripts/admin/import_brand_data.py \
     --file /app/data/pppl_brand.json \
     --admin-email rkremen@pppl.gov
   ```

### Can't connect to database

```bash
docker compose logs db
docker compose exec db psql -U tales -d tales -c "SELECT 1"
```
